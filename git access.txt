Usar o Git

Passo 1
Abra o GITHUB e acesse o link do projeto acesse o GITHUB e click
no link clone / download

Passo 2
Abre o CMD e executa em um diretorio de trabalho a linha 

C:\Users\a1420879>git clone + link do projeto https://github.com/DjRudPHD/Projeto1Web.git

Passo 3
Acesse seu projeto local apos a clonagem

C:\Users\a1420879>cd projeto1web

Passo 4
Inicialize seu projeto

C:\Users\a1420879\Projeto1Web>git init

Passo 5
Verifique o Status

C:\Users\a1420879\Projeto1Web>git status

Comece a editar e a fazer alterações nos arquivos

Passo 6 
Adicione os arquivos que vc alterou

C:\Users\a1420879\Projeto1Web>git add autores.php
Voce pode usar . para add todos os arquivos

Passo 7 
Verifique o Status

C:\Users\a1420879\Projeto1Web>git status

Passo 8
Commite as alterações e add os comentarios entre aspas

C:\Users\a1420879\Projeto1Web>git commit -m "Add Uma Linha"

Passo 9
Envie o arquivo para o GITHUB

C:\Users\a1420879\Projeto1Web>git push origin master
